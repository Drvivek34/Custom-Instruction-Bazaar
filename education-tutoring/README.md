## How entries are stored
Each instruction set lives in **its own sub-folder** inside this category, named in `kebab-case`:

```
education-tutoring/
└── <entry-name>/
    ├── README.md     # what it is, source, author, license, link
    └── ...           # the actual files (full copy)
```
