# 🛒 Custom Instruction Bazaar

> Curated system prompts and custom instructions for LLMs, by category.

[![Mega AI Bazaar](https://img.shields.io/badge/🌐_Mega_AI_Bazaar-browse_all-6C5CE7)](https://drvivek34.github.io/Mega-AI-Bazaar/) [![Submit](https://img.shields.io/badge/➕_Submit-a_custom_instructions-2ecc71)](https://github.com/Drvivek34/Custom-Instruction-Bazaar/issues/new/choose) [![License](https://img.shields.io/badge/License-MIT-blue)](LICENSE)

Custom Instruction Bazaar is a categorized library of **system prompts and custom instructions** — the persona/behavior packs you drop into ChatGPT, Claude Projects, Gemini Gems, and more. Standouts live in **featured/**.

## 🗂️ How it's organized
Every custom instructions lives in its **own sub-folder** inside a category, with a `README.md` recording its **source, author, and original license** (a full copy of the files is kept). See **[CONTRIBUTING.md](CONTRIBUTING.md)**.
